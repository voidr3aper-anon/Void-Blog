# GFW Bypass

This directory contains articles about bypassing internet censorship and the Great Firewall.

## Articles in This Category

- [Complete Guide to Bypassing the Great Firewall (GFW)](complete-bypass-guide/)

## Adding New Articles

To add a new article to this category:

1. Create a new directory: `_posts/gfw-bypass/your-article-slug/`
2. Add `index.md` file with frontmatter
3. Create `images/` subdirectory for article images
4. Follow the frontmatter template in existing articles

## Frontmatter Template

```yaml
---
layout: post
title: "Your Article Title"
date: {% now %} 
last_modified_at: {% now %}
author: "Your Name"
author_link: "https://github.com/yourusername"
categories: [GFW Bypass]
tags: [tag1, tag2, tag3]
excerpt: "Brief 1-2 sentence description"
---
```
