# Posts Directory Structure

This directory contains all blog posts organized by category.

## Structure

```
_posts/
├── network-monitoring/     # Network monitoring and diagnostics
├── privacy-security/       # Privacy, security, and anonymity
├── gfw-bypass/            # GFW bypass techniques
└── network-analysis/      # Protocol analysis and traffic inspection
```

## Creating New Posts

1. Choose or create appropriate category directory
2. Create post directory: `category/post-slug/`
3. Add `index.md` with proper frontmatter
4. Add images to `category/post-slug/images/`

See category README files for specific guidelines.

## URL Structure

Posts will be accessible at: `/category/post-slug/`

Example: `/network-monitoring/monitor-internet-connection/`
